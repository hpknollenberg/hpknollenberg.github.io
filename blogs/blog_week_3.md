# Henry Knollenberg's Week 3 Blog

## What are your thoughts about the roles HTML, CSS, and JavaScript play in the process of rendering content and providing a user experience?
I know that HTML is doing things, but after learning CSS and JavaScript, it feels like rather than doing things that it really just provides a base for other code to do things. 
It is interesting to me how the three work together to form one product. Before this bootcamp, I was not aware that different languages of code interacted with one another. It's also interesting how each seem to have a very specific purpose: HTML as the base display, CSS as the styling, and JavaScript as the logic. 

## What are some objects that your blog incorporates?
My blog has a header (the title), an image, an aside, and the list within that aside, which also includes links to each blog post.
